#Code Book
This code book sumarizes these variables of the `tidydata.txt`.

The data indicates the average of each measurements for each activity and each subject.
## Identifiers

* `subject` : Each identifies the subject who performed the activity for each window sample. Its range is from 1 to 30. 
* `activity` - Each identifies the activity name.
